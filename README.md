# Online Marketplace
Niewielki projekt na przedmiot Bazy Danych  
Online Marketplace umożliwa kupno sprzedaż i wymianę przedmiotów wirtualnych. 
Użytkownik ma własny ekwipunek z przedmiotami oraz konto z wirtualną walutą.

## Użyte techonologie
Baza danych - MongoDB  
Backend - NodeJS  
Frontend - Angular
